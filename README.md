## 开发 h5

常用的调试模式

```sh
yarn
yarn serve
```

## 开发微信小程序

下载安装 <https://dcloud.io/hbuilderx.html> <https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html>

微信开发者工具-设置-通用设置-安全-服务端口：打开

在 HbuildX 中 导入项目，运行-运行到小程序模拟器-微信开发者工具

然后在微信开发者工具中打开项目就行了
